# TEAMx Hackathon 2026: Reanalysis Group

The goals of this hackathon group are two-fold:
- To reformat wind lidar observations to be suitable for data assimilation in ICON (conversion to raw observation file format, ROF)
- To interpolate model data (ICON/MetUM) to be suitable for comparison with wind lidar observations

The group focussed on two convection IOPs:
- IOP 7: 29 June 2025
- IOP8: 30 June 2025

This respository is collecting our code and results. Hopefully, they provide a useful starting point for you to perform exciting analyses.
